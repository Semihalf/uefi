Notes about installing Buildbot on chad-pci under squeeze

1) aptitude install buildbot
	Buildbot home dir: chad-pci:/var/lib/buildbot
2) Change buildbot shell and set password
	Password: My normal one
3) Setup ssh keys and authorized hosts
4) Create a sample master:
	buildbot create-master master
5) Create two slaves:
	buildbot create-slave slave1 chad-pci:9989 chad-pci-debian64-slave1 caveo123
	buildbot create-slave slave2 chad-pci:9989 chad-pci-debian64-slave2 caveo123
6) Run update-buildbot.sh to copy over correct master file
7) Edit /etc/default/buildbot

creese@chad-pci:~$ cat /etc/default/buildbot
# buildbots to manage
# add a new set of variables for each buildbot to start

BB_NUMBER[0]=0                  # index for the other values; negative disables the bot
BB_NAME[0]="bdk-master"         # short name printed on startup / stop
BB_USER[0]="buildbot"           # user to run as
BB_BASEDIR[0]="/var/lib/buildbot/master"                # basedir argument to buildbot (absolute path)
BB_OPTIONS[0]=""                # buildbot options
BB_PREFIXCMD[0]="nice -n 19"            # prefix command, i.e. nice, linux32, dchroot

BB_NUMBER[1]=1                  # index for the other values; negative disables the bot
BB_NAME[1]="bdk-slave1"         # short name printed on startup / stop
BB_USER[1]="buildbot"           # user to run as
BB_BASEDIR[1]="/var/lib/buildbot/slave1"                # basedir argument to buildbot (absolute path)
BB_OPTIONS[1]=""                # buildbot options
BB_PREFIXCMD[1]="nice -n 19"            # prefix command, i.e. nice, linux32, dchroot

BB_NUMBER[2]=2                  # index for the other values; negative disables the bot
BB_NAME[2]="bdk-slave2"         # short name printed on startup / stop
BB_USER[2]="buildbot"           # user to run as
BB_BASEDIR[2]="/var/lib/buildbot/slave2"                # basedir argument to buildbot (absolute path)
BB_OPTIONS[2]=""                # buildbot options
BB_PREFIXCMD[2]="nice -n 19"            # prefix command, i.e. nice, linux32, dchroot

8) Start the buildbots
	/etc/init.d/buildbot start

9) Status page is at http://chad-pci:8010
